# Pillow Service

Microservicio FastAPI para crear fichas educativas A4 con imágenes y texto. Diseñado para generar material educativo infantil combinando ilustraciones generadas por IA con cuentos.

## Características
- Formato A4 (2480x3508px, 300 DPI)
- Composición automática de imagen + texto
- Título centrado con estilo
- Texto con sangría y formato profesional
- Borde decorativo opcional

## Tecnologías
- FastAPI
- Pillow (PIL)
- Python 3.11
